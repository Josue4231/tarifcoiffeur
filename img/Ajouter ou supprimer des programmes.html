Pour changer de lecteur (disque dur ou emplacement d'installation) sur Xbox Game Pass pour PC, voici les étapes à suivre :

✅ Changer le lecteur par défaut pour les installations futures

  1. Ouvre l’application Xbox sur ton PC.

  2. Clique sur ta photo de profil (en haut à droite), puis va dans Paramètres.

  3. Clique sur l’onglet Général.

  4. Dans la section "Changer le lecteur par défaut", tu verras un menu déroulant : choisis le lecteur que tu veux utiliser par défaut.

  5. Ensuite, clique sur "Changer l’emplacement d’installation".

✅ Changer le lecteur pour un jeu spécifique (au moment de l'installation)

  1. Va sur la page du jeu que tu veux installer.

  2. Clique sur Installer.

  3. Une fenêtre apparaîtra avec un menu déroulant pour choisir le lecteur : sélectionne celui que tu veux (ex. D:\ ou E:...).

  4. Lance l’installation.

💡 Attention : tous les jeux ne peuvent pas être déplacés après installation. Il vaut donc mieux choisir le bon lecteur avant d’installer.

✅ Déplacer un jeu déjà installé (si pris en charge)

  1. Ouvre les Paramètres Windows (Win + I).

  2. Va dans Applications > Applications installées.

  3. Trouve ton jeu dans la liste.

  4. Clique sur les trois points à droite du jeu, puis "Déplacer".

  5. Choisis un autre lecteur si c’est disponible.

⚠️ Si l'option "Déplacer" est grisée, c’est que le jeu ne supporte pas le déplacement.

Si tu veux, je peux t’aider à choisir quel lecteur utiliser en fonction de ton espace disque disponible. Tu peux aussi me dire si tu es sur Windows 10 ou 11 pour adapter les étapes.

Si aucun autre lecteur n’apparaît dans le menu déroulant de l'application Xbox ou dans les paramètres Windows, cela signifie généralement que :

❌ Windows ne détecte qu’un seul lecteur valide pour les installations de jeux.

Voici quelques raisons possibles et solutions :

✅ 1. Vérifie que le second lecteur est bien connecté et reconnu

  • Ouvre l’explorateur de fichiers (Win + E)

  • Vérifie si un autre disque (ex. D:, E:...) est visible

  • Si non : il faut peut-être le brancher, l’initialiser, ou le formater

➤ Pour initialiser ou formater un disque :

  1. Clique droit sur le bouton Démarrer → Gestion des disques

  2. Vérifie si un autre disque apparaît en “Non alloué”

  3. Clique droit dessus → Nouveau volume simple → choisis une lettre (D, E...) → Formater en NTFS

  4. Une fois fait, retourne dans l’application Xbox et réessaye

⚠️ Attention : le formatage efface toutes les données du disque concerné.

✅ 2. Le lecteur est détecté mais pas configuré pour les apps

Windows peut bloquer un lecteur s’il n’est pas configuré pour recevoir des applications.

Pour corriger ça :

  1. Ouvre Paramètres → Système → Stockage

  2. Clique sur "Emplacements d’enregistrement" (ou "Paramètres de stockage avancés" > "Modifier l’emplacement d’enregistrement du nouveau contenu")

  3. Sous "Les nouvelles applications seront enregistrées dans :", choisis ton autre lecteur

  4. Clique sur "Appliquer"

Ensuite, relance l’application Xbox et réessaye.

✅ 3. Réparer ou réinitialiser l’application Xbox et Gaming Services

Si malgré tout, rien ne s'affiche :

Réinitialiser Xbox :

  • Paramètres → Applications → Xbox → Options avancées → Réinitialiser

Réinstaller Gaming Services :

  1. Ouvre PowerShell en administrateur (clic droit sur Démarrer > Windows PowerShell (admin))

  2. Copie-colle cette commande :

    powershell
    get-appxpackage Microsoft.GamingServices | remove-AppxPackage -allusers
start ms-windows-store://pdp/?productid=9MWPM2CQNLHN

  3. Cela va rouvrir la page des Gaming Services dans le Microsoft Store : réinstalle-les

  2. Cherche Valorant

  3. Clique → Désinstaller
